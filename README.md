# AI소상공인 HP시안 (handa-hp)

노베이스 중장년 소상공인을 위한 AI 홈페이지 시안 생성 프로젝트.

- 저장소/폴더명(슬러그): `handa-hp`
- 배포 시 주소 예시: `handa-hp.vercel.app` (커스텀 도메인 연결 시에도 동일 슬러그 사용 권장)

## 진행 순서
1. 개발 환경 통일 (D:\GitHub\handa-hp)
2. 프로젝트 목적 정의 → `PROJECT_BRIEF.md`
3. Figma 와이어프레임 설계 → `references/wireframe.png`
4. 프로젝트 문서 작성 → `AGENTS.md`, `PROJECT_BRIEF.md`, `UI_REFERENCE.md`, `API_CONTRACT.md`
5. Hermes에게 분석 + 구현 계획 요청 (코드 수정 없이)
